# E-commerce Product Analysis Revenue Churn and Customer Behavior
Анализ e-commerce данных для выявления факторов, влияющих на выручку и churn.  Проведены анализ RFM, сезонность, возвраты и LTV. Обнаружено снижение выручки после сентября, высокий уровень возвратов и отсутствие явных факторов оттока.

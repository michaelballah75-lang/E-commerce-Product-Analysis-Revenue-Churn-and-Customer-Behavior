# E-commerce Product Analysis Revenue Churn and Customer Behavior
Анализ e-commerce данных для выявления факторов, влияющих на выручку и churn.  Проведены анализ RFM, сезонность, возвраты и LTV. Обнаружено снижение выручки после сентября, высокий уровень возвратов и отсутствие явных факторов оттока. 

Цели: 
Определить драйверы выручки
- Проанализировать LTV клиентов
- Исследовать churn
- Выявить сезонность
- Оценить возвраты

Ключевые инсайты
- Выручка снижается после сентября (сезонность)
- Высокий уровень возвратов во всех категориях
- Поведенческие метрики (RFM) не объясняют churn
- Выручка зависит от небольшой группы клиентов

Рекомендации
- Улучшить карточки товаров и снизить возвраты
- Запускать маркетинговые кампании в Q4
- Расширить сбор данных для анализа churn
- Сфокусироваться на удержании high-value клиентов
